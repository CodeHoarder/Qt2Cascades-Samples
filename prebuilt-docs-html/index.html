Qt Cascades Examples

The following example projects are more or less straightforward adaptations of the well-known Qt example projects to the BB10 platform. The BB10 platform differs from the Qt framework in several areas, and the original Qt examples are adapted to those differences.

A major difference of the BB10 platform is the UI framework, which is called Cascades. Cascades is based on QtQuick, but provides an alternative set of UI components. The following example projects are all based on the BB10 Cascades UI framework. Other differences, in non-UI portions of the BB10 platform, are also reflected in the adaptation of the original Qt example projects.

  • Multithreading
    • Semaphores
    • Wait Conditions
    • Queued Custom Type
  • Network
    • Broadcast Receiver
    • Broadcast Sender
    • Network Download
    • Network Download Manager
    • FTP
    • HTTP
    • Multicast Receiver
    • Multicast Sender
    • Secure Socket Client
  • SQL
    • SQL Widget Mapper
  • State Machine
    • Factorial
    • Ping Pong States
  • Tools
    • I18N
    • Regular Expressions
  • XML
    • DOM Bookmarks
    • SAX Bookmarks
    • QXmlStream Bookmarks
    • RSS-Listing
    • XML HTML Info
    • XML Stream Lint
